# Smart Expense Tracker

Web application to track personal expenses. Users can register, login, add/view expenses, and see monthly reports. Built with JSP, Servlets, and MySQL.

## Features
- User registration and login
- Add, view, edit, and delete expenses
- Monthly expense reports
- Session handling and authentication

## Technologies
- Frontend: HTML, CSS, JSP
- Backend: Java Servlets
- Database: MySQL
- Connectivity: JDBC
